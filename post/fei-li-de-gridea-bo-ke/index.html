“费力的” Gridea 博客

2020-04-10

昨天不小心在 Gridea 配置里面把令牌（也就是 GitHub 上创建的 token ）给改掉了

今天又花了好久来把这个东西迁移到了另外一个账号上

同时也把评论的功能加上了，是基于 Gitalk 插件的

这些看似简简单单的东西我来回鼓捣了半天多

有时候看似一个特别简单的问题每个人操作起来都会是千奇百怪的结果

所以能写一个好的教程是真的不容易的

一个比较好的方式就是基于问题的时时沟通，把自己的问题向相关的牛人请教

这其实也是我们在上学时候问老师的样子

但是自己不能永远做学生

要学习道理和知识，

在面对问题、解决问题时，我们就是在快速的成长

要有意识地思考到这一点

千万别让自己在问别人问题的环境中成为一个巨婴
